# Stack

## Stack의 특징
- #### 후입선출(LIFO - Last In First Out)이다.
- #### 예) 인터넷 브라우저 뒤로가기, Ctrl + z
- #### 가장 먼저 넣은 데이터가 가장 나중에 나오게 된다.
- #### 가장 나중에 넣은 데이터가 가장 일찍 나오게 된다.

## Stack에서의 중요한 연산
- #### push()
  - data를 넣는 연산(접시쌓기 개념으로)
- #### pop()
  - data를 꺼내오면서 data를 가져오는 연산
- #### top(), peek()
  - data를 그래도 둔 채로 data를 가져오는 연산
